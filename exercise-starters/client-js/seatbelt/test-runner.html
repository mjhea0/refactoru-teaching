JS22.9: Seatbelt

Test Runner

This is a "test runner," a simple page whose only job is to launch your tests. You can hit refresh to reload the tests.

Three scripts are included:

  • src.js - Your source code. This is the code that will be running on your website, and you wish to test. In this exercise, the isEven function has already been written for you.
  • test-framework.js - Your testing library, such as Jasmine or Mocha. Here we are using a (very) tiny home-made test framework. Do not edit this file.
  • spec.js - A separate file that calls methods from the test framework on your source code in order to "assert" that your source code behaves as expected. Write your unit tests in this file.

Open console to see results.
